### Import

```js
import DocumentSummary from '@ltht-react/document-summary'
```

```jsx

```

### Properties

| Prop        | Required | Default | Type                | Description                     |
| :---------- | :------- | :------ | :------------------ | :------------------------------ |
| `documents` | No       |         | DocumentReference[] | An array of document references |
