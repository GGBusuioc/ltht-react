# Spinner Icon

### Import

```js
import SpinnerIcon from '@ltht-react/spinner-icon'
```

### Usage

```jsx
<SpinnerIcon />
```

### Properties

| Prop   | Required | Options                    | Type   | Description |
| :----- | :------- | :------------------------- | :----- | :---------- |
| `size` | Yes      | 'small', 'medium', 'large' | string | icon size   |
